# Java Hash-Map

## Requirements

Install Java 11 or a newer version.

## Run

```
java HashMap.java 
{1=number one, 2=number two, 3=number three}
{1=number one, 2=number two, 3=number threeeeee}
{2=number two, 3=number threeeeee}
2 -> number two
```
